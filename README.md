# Frontend - Hennesy Project

## Requirements
- Node.js 18+

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure `.env`:
   - Ensure `VITE_API_BASE_URL` points to your backend (e.g., `http://localhost:8000`).

## Running
Start the development server:
```bash
npm run dev
```
App runs at `http://localhost:5173`.
